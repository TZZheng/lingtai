How to regenerate

Agent-assisted (Codex / Claude-style)

/graphify .

Headless CLI (needs an LLM backend for docs/papers/images)

export GEMINI_API_KEY=...
graphify extract . --backend gemini

Useful follow-ups

graphify query "Why does contains() connect so many code and test nodes?"
graphify explain "Preset"
graphify path "Agent Operating System" "Filesystem-only IPC"
graphify export html

Recommended workflow

  1. Generate the graph with /graphify . or graphify extract . --backend gemini.
  2. Open graphify-out/graph.html for the high-level community map.
  3. Read graphify-out/GRAPH_REPORT.md for god nodes and leads.
  4. Use query / explain / path to investigate architecture questions before broad changes.
  5. Verify inferred and ambiguous edges against source files before treating them as design evidence.
